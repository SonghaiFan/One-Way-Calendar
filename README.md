# One-Way Calendar / 单向历

[English](#english) | [中文](#chinese-chinese)

## English

### Introduction

A minimalist, web-based implementation of the famous OWSPACE (单向空间) "One Way Street" calendar. This application brings the physical tear-off calendar experience to your browser with a clean, paper-like aesthetic.

### Features

- **Daily Wisdom:** Displays the unique calendar page for each day from the OWSPACE collection.
- **Realistic UI:** Designed to look like a physical calendar with binding rings, paper texture, and stacked pages effect.
- **Navigation:**
  - Browse past and future dates using on-screen controls.
  - Keyboard support: Use `Left Arrow` and `Right Arrow` keys to navigate.
  - "Return to Today" button for quick access.
  - Date picker to jump to any specific date.
- **Responsive Design:** Works beautifully on both desktop and mobile devices.

### How to Use

This is a standalone HTML application. No build process or installation is required.

1. Clone this repository or download the `index.html` file.
2. Open `index.html` in any modern web browser.

### Tech Stack

- **React 18:** For UI components and state management.
- **Tailwind CSS:** For styling and responsive design.
- **Babel Standalone:** To compile JSX directly in the browser.
- **Lucide React:** For iconography.

### Credits

- Calendar content and images are property of **OWSPACE (单向空间)**.
- Image proxy service provided by `wsrv.nl`.

## 中文 (Chinese)

### 简介

这是一个基于 Web 的极简主义“单向历”应用。它将单向空间（OWSPACE）经典的纸质撕历体验带到了浏览器中，拥有干净、拟真的纸质风格界面。

### 功能特点

- **每日宜忌:** 展示单向历每日独特的页面内容。
- **拟真界面:** 设计模仿实体日历，包含金属环扣、纸张纹理和堆叠效果。
- **浏览导航:**
  - 使用屏幕下方的控制栏浏览过去或未来的日期。
  - 键盘支持: 使用 `左箭头` 和 `右箭头` 键进行切换。
  - “回到今天”按钮可快速返回当前日期。
  - 日期选择器可跳转至任意指定日期。
- **响应式设计:** 在桌面端和移动端设备上均有良好体验。

### 如何使用

这是一个独立的 HTML 应用程序，无需构建过程或安装依赖。

1. 克隆此仓库或下载 `index.html` 文件。
2. 直接在任意现代网页浏览器中打开 `index.html` 即可。

### 技术栈

- **React 18:** 用于 UI 组件和状态管理。
- **Tailwind CSS:** 用于样式和响应式设计。
- **Babel Standalone:** 用于在浏览器中直接编译 JSX。
- **Lucide React:** 提供图标支持。

### 致谢

- 日历内容和图片版权归 **单向空间 (OWSPACE)** 所有。
